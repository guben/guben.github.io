Blog上发布文章步骤

防止自己遗忘，将步骤写一下

  1. 首先在blog目录下打开CMD

  2. hexo new name #创建新文章

  3. 找到文章，使用markdown编辑

  4. 部署

    hexo generate #生成
    hexo server #启动服务预览
    hexo deploy #部署发布
    hexo clean #清除缓存

  5. 简写

    hexo n “我的博客” == hexo new “我的博客” #新建文章

    hexo p == hexo publish

    hexo g == hexo generate#生成

    hexo s == hexo server #启动服务预览

    hexo d == hexo deploy#部署
